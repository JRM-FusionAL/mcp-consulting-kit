## One-Line Description

Privacy-first MCP gateway with policy enforcement and audit-ready operations for production deployments.

## Full Description

FusionAL is a self-hosted MCP governance gateway for teams moving from MCP prototypes to production.

It is built for organizations that need control and visibility over AI tool execution, especially in privacy-sensitive and regulated environments.

Core capabilities:

- policy enforcement over tool access and runtime behavior
- audit-ready activity visibility and export workflows
- Docker-first self-hosted deployment
- cross-platform reliability (Windows, Linux, macOS)

FusionAL also includes implementation patterns used in real consulting engagements, where governance requirements are captured and converted into enterprise feature priorities.

## Feature Bullets (Smithery UI)

- Self-hosted MCP gateway
- Policy profiles (`strict`, `balanced`, `dev`)
- Audit export workflow (JSON/CSV)
- Tenant/API key controls + rotation patterns
- Rate-limit presets for governed operations
- Windows/Linux/macOS support
